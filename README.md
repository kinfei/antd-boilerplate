# antd-boilerplate

Boilerplate for using antd pro dashboard with light / dark mode switch

## How to Run

```bash
$ git clone git@github.com:kinfei/antd-boilerplate.git
$ cd antd-boilerplate
$ yarn install
$ yarn start
```

## How to Build

```bash
$ yarn build
```

## Prepare for Deploy

```bash
$ yarn deploy
```
