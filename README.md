# antd-boilerplate

Boilerplate for using antd pro dashboard

[Link to Demo](https://fir-boilerplate-f7d84.web.app/)

## How to Run

```bash
$ git clone git@github.com:kinfei/antd-boilerplate.git
$ cd antd-boilerplate
$ yarn install
$ yarn start
```

## How to Build

```bash
$ yarn build
```

## Prepare for Deploy

```bash
$ yarn deploy
```
